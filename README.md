# -iTunes-Music-Store-Data-Analysis-SQL-
This project analyzes the iTunes music store database to uncover insights about customer purchasing behavior, music popularity, and revenue trends. The analysis was performed using Microsoft SQL Server Management Studio for querying and Tableau Desktop for interactive dashboard creation.

🎯 Objectives :- 
The main objectives of this project are:
Analyze customer purchasing behavior
Identify the most popular music genres and artists
Track revenue trends over time
Rank customers based on spending
Create an interactive dashboard for business insights

🛠Technologies Used :-
SQL (Data Analysis)
Microsoft SQL Server Management Studio
CSV Data Files

📊 Key Insights
From the analysis, the following insights were observed:
A small number of customers generate a large portion of revenue.
Certain music genres dominate customer purchases.
Some artists consistently generate high revenue.
Revenue distribution varies significantly across countries.

💡 Business Recommendations
Based on the analysis:
Promote top-performing artists and genres
Provide personalized recommendations to loyal customers
Focus marketing efforts in high-revenue countries
Create curated playlists for popular tracks
